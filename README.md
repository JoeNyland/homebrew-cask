# homebrew-cask

A collection of Casks, installable via [Homebrew][homebrew].

## Usage

Tap the repo and install a cask:

    brew tap joenyland/cask
    brew install <cask>

## Troubleshooting

Please check the main [Homebrew troubleshooting guide][homebrew-troubleshooting] and then [open an issue in this tap][issues].

[homebrew-troubleshooting]: https://github.com/Homebrew/brew/blob/master/docs/Troubleshooting.md
